ДЛЯ ПИЦЦЫ

Гофролотки для пиццы- это один из типов самосборных гофрокоробов. Главная особенность упаковки для горячей пиццы в том, что она должна одновременно и сохранять тепло, и быть достаточно влагоустойчивой, чтобы не размокать от пара.

Характеристики:

Изготавливается по размерам заказчика как из бурого, так и из белого гофрокартона.

Марки гофрокартона: T-23; Т-24; Т-25; Т-26

Тип гофра: B, C

Толщина: от 3,0 до 3,8 мм

Возможно нанесение печати различных цветов.

Всегда в наличии имеются следующие размеры: 

1. П (250х250х40) C Т23 Ш Б №25 (0,218 м²)

2. П (310х310х40) C Т23 Ш Б №31 (0,293 м²)

3. П (330х330х40) C Т23 Ш Б №34 (0,384 м²)

4. П (360х360х40) C Т23 Ш Б №36 (0,387 м²)

5. П (380х380х40) C Т23 Ш Б №38 (0,457 м²)

6. П (400х400х40) C Т23 Ш Б №40 (0,496 м²)

7. П (400х400х45) C Т23 Ш Б №45 (0,479 м²)
